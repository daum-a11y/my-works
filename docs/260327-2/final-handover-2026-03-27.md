- 관리자 요약

- 전체 업무검색

- 현재 1차 대시보드의 2개 테이블은 유지
- 2차 대시보드의 `내 업무보고 작성현황` 캘린더 병합 완료
- 2차 대시보드의 우측 안내 이미지 병합 완료
- 캘린더 날짜 클릭 시 개인 업무보고 입력 화면으로 날짜 전달 완료
- 브라우저 재검증 완료
