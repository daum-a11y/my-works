# 레거시 정리 대상

다음 영역은 신규 Supabase 스키마에 생성하지 않습니다.

## 기능 제외

- 대시보드 추천 모니터링
- 앱 운영정보
- 알림
- 호출처 불명 `manager_*`

## 신규 스키마 미생성 대상

- `APPINFO_TBL`
- `NOTI_TBL`
- `AGITNOTI_TBL`
- `AGITNOTI_OPT_TBL`

## 정리 원칙

- 위 테이블은 1차/2차 신규 React 이관 범위에서 제외합니다.
- 레거시 MySQL에서는 정리 후보로 문서화만 하며, 실제 삭제 계획은 별도 운영 검토 후 수행합니다.
